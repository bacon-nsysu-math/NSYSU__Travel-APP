# NSYSU__Travel-APP

We are Group 4 from National Sun Yat-sen University - Business Building Software Design. The following is our final project APP. We will use "streamlit_folium" in Python to present our "Kaohsiung Travel Itinerary APP Design".

## Quick Start

Simply download the entire repository and execute `STARTUP.cmd` to automatically launch the Travel-APP.

### Usage Steps:
1. Download the entire project repository
2. Locate and double-click the `STARTUP.cmd` file
3. The system will automatically start the Travel-APP

## Features

- Interactive Kaohsiung travel itinerary planning
- Built with Python using streamlit_folium
- User-friendly interface for exploring Kaohsiung attractions
